## Recommendations

### Primary benchmark: **Yelp**
- 268,653 users × 109,325 merchants (full k-core)
- Explicit star ratings → cleanest recommendation ground truth
- 5 temporal slices for SelfGNN self-supervised learning

### Implicit feedback benchmark: **Finance**
- 2,000 users × 74,831 merchants — sparse realistic graph
- 10 annual time slices (2010–2019)
- Transaction amount as implicit preference proxy

### Controlled experiments only: **Synthetic**
- 48 merchants → density 4.6% — near-complete graph
- GNN message passing provides minimal signal in dense graphs
- Useful for scalability / ablation studies only
